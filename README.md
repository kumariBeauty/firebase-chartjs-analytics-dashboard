# Firebase Chart.js Analytics Dashboard

A real-time analytics dashboard built using **React**, **Firebase Firestore**, and **Chart.js**.  
The application displays dynamic **line, bar, and pie charts** that update automatically when data changes in Firebase.

## Features
- Real-time data visualization using Firebase Firestore
- Line, Bar, and Pie charts with Chart.js
- Responsive and clean dashboard UI
- Live updates without page refresh

## Tech Stack
- React
- Firebase Firestore
- Chart.js
- JavaScript, HTML, CSS
